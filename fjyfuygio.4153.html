📋 Common Fixes

Error: "Failed to fetch"

  1. script.google.com → Deploy → New deployment
  2. "Who has access" → Anyone
  3. New /exec URL → Settings में paste करें

Image Not Showing on Profile

  1. Upload via admin panel (file upload) — Drive URL auto-generates
  2. If using URL: must be a public direct image link
  3. Google Drive: File → Share → Anyone with link → Copy file ID → use lh3.googleusercontent.com/d/FILE_ID

Data Not Showing (Address, Type)

  1. Run initializeSheets() in Apps Script (adds payment_link column)
  2. Make sure code.gs is the latest v7 version
  3. Re-save the client after update
